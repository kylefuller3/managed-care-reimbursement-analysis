# Managed Care Reimbursement & Utilization Trend Analysis

## Project Overview

This project analyzes simulated managed care claims data to evaluate reimbursement performance, utilization trends, denial patterns, payer behavior, and service-line financial performance.

The analysis was built in Microsoft Excel using structured tables, formulas, summary views, and dashboard-style reporting to support healthcare financial and operational decision-making.

## Business Objective

The goal of this project is to identify reimbursement and utilization trends that could help a healthcare organization:

- Evaluate payer performance
- Monitor reimbursement rates
- Identify high-denial areas
- Compare service-line performance
- Support managed care decision-making
- Improve reporting accuracy and financial visibility

## Dataset

The workbook contains simulated managed care claim records across multiple payers, service lines, and reporting periods.

Key fields include:

- Payer
- Service line
- Quarter
- Charges
- Reimbursement
- Denials
- Utilization
- Reimbursement rate
- Financial and operational summary metrics

## Tools Used

- Microsoft Excel

## Key Analysis Areas

### Reimbursement Analysis

The workbook evaluates reimbursement performance across payers and service lines to identify which areas produce stronger or weaker payment outcomes.

### Utilization Trends

The analysis reviews service utilization patterns over time to identify volume changes and demand trends.

### Denial Analysis

Denial patterns are reviewed to identify payer and service-line areas that may require closer monitoring or process improvement.

### Payer-Service Line Performance

The project compares payer and service-line relationships to highlight areas with higher financial risk or stronger reimbursement performance.
